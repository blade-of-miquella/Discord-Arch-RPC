# Arch Linux Discord RPC

Shows "using arch btw" in your Discord status. 
Supports automatic startup for **Hyprland** and standard XDG environments (GNOME, KDE, etc).

## Installation

Run the following commands in your terminal:

```bash
# Clone the repository
git clone [https://github.com/blade-of-miquella/Discord-Arch-RPC.git](https://github.com/blade-of-miquella/Discord-Arch-RPC.git)
cd Discord-Arch-RPC
chmod +x install.sh
./install.sh
